About me

Hello! My name is Deji and I enjoy creating things that could make people’s lives easier on the internet. My interest in web development started back in 2021 when I noticed the ease it brings to life and how fun it will be to be actually be the one creating it, not just the one at the receiving end. Turns out it won’t be easy but here I am, getting better! My first attempt was paying people to teach me which didn’t end well so I started self-teaching via YouTube videos untilI I came across I4GZuri free Tech Training which has really helped me improve alot. Fast forward to today, I can now confidently create a web app using HTML, CSS and
JS and I am also conversant with React and Tailwindcss. Hoping to add PHP and Wordpress by the end of the year.
